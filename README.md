
# Eum -  지역 공동체 활성화 플랫폼


### 1. 프로젝트 소개
‘햇살마을’은 ‘시간은행’의 개념을 바탕으로, 지역 주민들이 서로의 시간과 재능을 나누는 형식으로 구성된다. 이를 통해 지역 공동체의 상생을 추구하며, 고립된 개인이나 지역에 필요한 자원을 제공할 수 있다.
